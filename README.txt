== Frequently Asked Questions ==
How we can use the \"Data Access\" plugin?
 
1) Once install the plugin, one option in the \"Data Access\" menu is shown in the  WordPress admin dashboard.
 
2) ON clicks this option then the database details are shown.
 
3) Click on the “Open Database” button, and MySQL will open.
